# 伴航 NavPal 隐私政策

**最后更新：2026-06-13**

## 信息收集

NavPal 扩展不收集任何个人信息。所有数据（包括书签、设置、偏好）均存储在您的本地浏览器中，使用 Chrome 的 storage API，不会传输到任何远程服务器。

## 数据使用

- 书签数据：仅用于在新标签页展示您保存的网站
- 语言偏好：仅用于界面语言切换
- 所有数据完全由您控制，仅在您的设备上使用

## 权限说明

| 权限 | 用途 |
|------|------|
| storage | 存储书签和偏好设置，数据保存在本地 |

## 第三方服务

NavPal 不使用任何第三方分析、跟踪或广告服务。

## 数据删除

卸载扩展时，所有本地数据将被自动清除。您也可以在扩展设置中手动清除数据。

## 联系方式

如有隐私相关问题，请通过 GitHub Issues 联系：
https://github.com/panris/navpal-extension/issues

## 政策更新

如更新此政策，将在扩展更新说明中通知用户。
